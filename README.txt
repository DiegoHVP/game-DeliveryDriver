## Jogo - Entregador de Mercadorias

Um jogo simples onde o objetivo é entregar mercadorias resolvendo problemas matemáticos, desenvolvido para o ensino fundamental. O jogador controla um personagem que deve pegar N caixas e entregá-las no caminhão, onde N representa a quantidade de caixas e a resolução de uma questão fornecida.

### Sons utilizados

- **Undertale**
  - Nyeh Heh Heh! (Intro) + Bonetrousle Extended
  - Confession
  - Last Goodbye
  
- **Cave Story**
  - Theme
  - Got Item! (Internal Percussion)
  - E outros efeitos sonoros

### Sprites

Na pasta `Player`, você encontrará o sprite do personagem sem a caixa.
Na pasta `PlayerBox`, você encontrará o sprite do personagem segurando a caixa.
Sinta-se à vontade para explorar e experimentar este projeto.
